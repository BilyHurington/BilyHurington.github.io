8-16比赛 T1解题报告

T1 奖金

1.1 序言

小H是一个很有钱的男人，所以他为了体现自己的富有并且回馈小弟对自己忠诚，决定给小弟们发奖金

1.2 题目描述

小H有n元钱，10^(10^10)个小弟。他要把这些钱全部发给他的小弟。他决定按照小弟对自己的忠诚度(每一个小弟的忠诚度不同)分发奖金，显然忠诚度高的小弟分到的钱不会低于忠诚度底的小弟。

每一个小弟分到的钱一定是非负整数，可以 有小弟没分到钱 小H现在想知道一共有多少种分发奖金的方案，两种方案被认为是不同的当且 仅当存在一个小弟在两种方案中得到的钱不同。

由于小H有很多社交活动，没有时间去计算方案数，所以他把这个问题丢给你来解决，输出的方案数对998244353取模

1.3 输入格式

从文件money.in中读入数据. 一行仅包含一个正整数n，表示小H拥有的钱

1.4 输出格式

输出到文件money.out中. 输出一行包含一个整数，表示分发奖金的方案数

1.5 样例1输入

4

1.6 样例1输出

5

1.7 样例1解释

一共有以下5种方案{1,1,1,1},{2,1,1},{3,1},{2,2},{4}

1.8 样例2输入

233

1.9 样例2输出

77164040

1.10 样例3输入

66666

1.11 样例3输出

526091567
